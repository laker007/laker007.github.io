React-Native

发表于 2017-02-05   |   分类于 React Native 初识   |     |   阅读次数

在慕课网上看到了很多前端的实战课程，都需要付费购买。我看了也是非常的心动，都是我想要学习的课程，可是一摸兜里没有钱呐。于是乎我就到处淘资料，心想肯定有有识之士购买了
课程之后把课程录下来的。正所谓功夫不负有心人，终于在一个叫做稀酷客的网站上面找到了免费的慕课网实战课程:贯穿全栈 React Native 开发 APP。利用下班之后的时间和春节
过年的空隙，算是囫囵吞枣的把视频基本撸完了，还差一章视频录像处理的没看。说实话，React Native 开发的基本流程走了一遍，半懂不懂。因为没有跟着一起做，所以很多的
要点没有完全掌握。视频作者 Scott 哥哥使用的是 React Native 的 ES5 写法。而 React Native 现在已经全面从 ES5 转向了 ES6。后面在写 RN 代码时，我会全部使用 ES6
的写法来做，算是对自己 ES6 的一个修炼。
昨天晚上，撸完视频，想着要不写个 tabbarIOS 底部导航栏练练手？还好用的是 MAC OS ，装环境遇到的坑一个没碰上，算是万幸了。按照官网的示例代码，费了老大的劲，
算是把底部导航栏给撸出来了。但是切换 tabbar 没有任何效果，明天再调试一下。算是一个不完美的开始。RN 搞起来！！！
